# Aether Advisor OS

Aether Advisor OS is a desktop-first wealth advisor operating console. This prototype turns the v5.0 vision document into a grounded product with client intelligence, meeting workflows, AI agent governance, compliance review, portfolio exposure, and practice management.

## Product Scope

- Advisor dashboard with household risk, meetings, approvals, and audit evidence.
- Client Hub with family map, accounts, goals, document status, and recommendations.
- Meeting Assistant with transcript signals and human approval gates.
- Agent Swarm Control with autonomy levels, confidence, reasoning, and pause controls.
- Compliance Shield with review queue, audit trail, and regulatory control coverage.
- Portfolio Manager with enterprise exposure, IPS exceptions, and scenario summaries.
- Team OS with capacity risk and delegation rules.
- Governance Settings for access, records, AI policy, and data sources.

## Implementation

- React 19, TypeScript, Vite.
- Mocked product data is in `src/data/aetherData.ts`.
- Core product types are in `src/types.ts`.
- UI state and workflows are in `src/App.tsx`.
